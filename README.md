# Starter Kit for Theme Development
Custom Shopify Theme - Shopify CLI compliant.

## Tech Stack
* NodeJS
* Bootstrap 5.x
* SASS 
* Webpack + PurgeCSS

### Images Lazy Loading
* Use native Lazy loading
* Fallback with [Vanilla lazyload](https://github.com/verlok/vanilla-lazyload)

### Dependencies
* Use package.json
* Use `import`
